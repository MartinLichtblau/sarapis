Sarapis API

A proxied Solr REST API to enable easy read-only access to any Solr instance.

Summary

Solr Search

GET /solr/{query}

Extended DisMax Syntax applies, refer to https://cwiki.apache.org/confluence/display/solr/The+Extended+DisMax+Query+Parser for additional information.
query Solr search query path string
page Pagination offset, e.g. 5 query integer
rows Number of results per page, e.g. 100 query integer
sort Sort function, e.g. id desc, price asc query string
rangeFilter Restrict results to field values in range, e.g. price,5,30 query string
matchFilter Restrict results to matching field values, e.g. author, tolkien query string
restrict Restrict results to specified fields, e.g. id,size,weight query string
groupBy Group results with the specified field, e.g. publisher query string
timeout Limit response time in milliseconds and get preliminary results, e.g. 10 query integer
